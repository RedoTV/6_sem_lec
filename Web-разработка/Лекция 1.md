## Вопрос 1. Введение в js 

Язык был разработан в NetScape Communications для браузера NetScape Navigator в декабре 1995 года на основе языка Java. 
В июне 1997 года был принят стандарт языка под названием ECMAScript. 

## Вопрос 2. Синтаксис JS. Основные понятия.

Язык чувствителен к регистру.
Для переносов строки используется '\\' (обратный слеш).
// - однострочный  комментарий
/\*\*/ - Многострочный комментарий
При описании переменных принято:
1) все имена строчными буквами
2) на стеке слов большая буква или подчеркивание
3) переменные и свойства называем именами существительными
4) массивы и коллекции называем именами существительными во множественном числе. 
5) функции и методы называем глаголами.
6) названия классов с прописной(большой) буквы.

JS - динамически типизированный язык программирования, интерпретатор определяет тип переменной по правой части.
Типы данных:
1) Bolean
2) String
3) Number
4) Object
5) Null
6) Undefined - если переменной var не присвоено значение то при обращении к ней выведется.
7) Symbol - используется для создания уникальных идентификаторов объекта
8) BigInt - используется для работы с большими числами, большими чем 2^53

Для определения типа переменной используется оператор $typeof$ 
! (not)  &&(and)
|| (or) ^(xor)

При сложении разных типов, если один из операндов строка, то второй автоматически преобразуется к строке. 
Унарный плюс осуществляет численное преобразование.

### Вопрос 3. Функции для работ с типами данных, ввод вывод данных.

parseInt(строка, основание) - преобразует указанную строку в целое число по указанному основанию системы счисления.
parseFloat(...) - преобразует указанную строку в число с плавающей точкой.
isNaN(значение) - служит для определения того, является ли значение числом.

### Вопрос 4. Условные операторы в JS

1) if else
2) Тернарный оператор - дня выполнения нескольких операторов по одному условию можно использовать запятую.
3) switch (value) { 
   case 'голубые' : alert('dsadad'); break; 
   default: alert('dsfdgdb'); 
   } 
   оператор break обязателен после каждого значения, если его не использовать, то будут выполнены все расположенные ниже операторы. Операторы блока default будут выполнены если ни одно из перечисленных значений не подходит.

### Вопрос 5. Операторы циклов

Оператор $for$ 
В циклах можно использовать $break$ и $continue$

Нужно динамически генерировать маркированные и нумерованные списки на веб странице в зависимости от введенных пользователем данных: вид списка и количество пунктов списка. В зависимости от ответа выводить теги либо маркированного либо нумерованного списка с необходимым количеством пунктов.
